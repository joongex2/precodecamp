ความเหมือน ความต่าง
<hr> & <br> เป็น empty tag hr จะขึ้นบรรทัดใหม่เพิ่มเส้นแล้วขึ้นบรรทัดใหม่ br แค่ขึ้นบรรทัดใหม่
<ul> & <ol> ไว้สร้าง list เหมือนกัน ul หน้า list แต่ละ list ขึ้นต้นด้วย bullet หรือ สัญลักษณ์ ส่วน ol ขึ้นต้นด้วยตัวเลขไล่ลำดับลงไป
<select> & <checkbox> & <radio button> select กับ radio button สามารถเลือกได้ค่าเดียว
checkbox กับ radio เป็น type ของ input tag		 checkbox เลือกได้หลายค่า ส่วน select, radio ได้ค่าเดียว
select tag ชนิดนึง แต่ checkbox และ radio เป็น type ของ input tag อีกที
checkbox ต้องกดลูกศรก่อนถึงโชว์ค่าทั้งหมด แต่ select, radio โชว์ ค่าทั้งหมดเลย
<div> & <span> มักใช้เพื่อกำหนด css/ จัด layout ให้ข้อความหรือ element ต่างๆ div เป็น block element คือจะขึ้นบรรทัดใหม่ต่อจาก element ก่อนหน้า span เป็น inline element คือไม่ขึ้นบรรทัดใหม่ต่อจาก element ก่อนหน้า (มักใช้กับข้อความสั้นๆในประโยค)
<script> & <style> เป็น non-empty tag script ใช้เพื่อ ใส่ javascript code ใน html (ใส่ใน body), style ใช้เพื่อใส่ css code ใน html (ใส่ใน head)
