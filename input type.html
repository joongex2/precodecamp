type ลักษณะ
button สร้างปุ่ม
checkbox สร้าง checkbox ที่ติีกได้มากกว่า 1 ค่า
color เป็นปุ่มกดแล้วขึ้น color picker เมื่อเลือกแล้วเก็บเป็น rgb #ฐาน16 6 หลัก
date เก็บเป็นค่า mm/dd/yyyy ซึ่งพิมพ์เปลี่ยนเองหรือ กด datepicker ด้านขวาได้
datetime-local เก็บเป็นค่า mm/dd/yyyy ซึ่งพิมพ์เปลี่ยนเองหรือ กด datepicker ด้านขวาได้ และเก็บ time ใน format 12:59 AM
email เป็น input มีการ validate ค่าให้เป็นไปตาม format ของ email (xxx@xxx.com) หากไม่ผ่านจะ submit ไม่ผ่าน
file เป็นปุ่ม Choose File กดแล้วสามารถเลือก file จาก device ได้
hidden จะไม่เห็น ช่อง input ซึ่งใช้เพื่อ ส่งค่า (สามารถกำหนด value ได้) ตอน form submit โดยที่ไม่แสดงให้ user เห็น
image ใช้เพื่อแทนปุ่ม submit ซึ่งกำหนดรูปเอง ของ form ได้และส่ง x, y ของการคลิ๊กในภาพ ด้วย
month เก็บค่าเป็น format December 2020 และสามารถใช้ datepicker ด้านขวาเลือกเดือนได้
number เก็บค่าตัวเลข ซึ่งบังคับให้พิมพ์แค่ตัวเลข หรือใช้ลูกศรด้านขวาสามารถปรับเพิ่มลดค่าได้ สามารถกำหนด max min ได้
password เก็บค่าซึ่งเวลาพิมพ์จะไม่โชว์ค่า (โชว์เป็น bullet สีดำแทน)
radio ใน tag input name เดียวกันจะเลือกค่าได้ค่าเดียว
range เป็น slider แนวนอน เก็บค่าตัวเลข มีปุ่มเพื่อเลื่อนซ้ายขวาได้ ซ้ายสุดเป็น min ขวาสุดเป็น max
reset เป็นปุ่มเพื่อ reset ค่าทุกอย่างใน form ทุกอย่างเป็น default value
search ใช้ในการพิมพ์ค่าเพื่อ search คล้ายๆ กับ input type text แต่มีกากบาทด้านขวา
submit ทำปุ่มใน form เพื่อใช้ในการส่งค่าไปยัง action ที่อยู่ใน form tag
tel ใช้เพื่อ validate ค่าให้ตรงกับ pattern ของมือถือ เช่น 123-45-678 ใช้ pattern="[0-9]{3}-[0-9]{2}-[0-9]{3}" หาก ผิด format จะ submit ไม่ผ่านและขึ้นว่า Please match the requested format.
text ใช้เพื่อเก็บค่า value อะไรก็ได้โดยพิมพ์เอา
time เก็บค่าเวลาใน format 12:59 AM มีปุ่มกากบาทเพื่อลบและขึ้นลงเพื่อเพิ่มลดค่าตอนเลือกช่องต่างๆ
url ใช้เพื่อเก็บค่าและ validate ค่าที่เป็น format url หากไม่ผ่านจะแสดง Please enter a URL
week ใช้เพื่อเก็บค่า week ใน ปีใดๆ format Week 51, 2020 มีกากบาทเพื่อ reset และใช้ week picker ได้
